# LABESK
> VIRUS

How to Use / Como Usar ?:

1. Tenha o Git e o gcc Instalado
2. Clone este Repositorio:
```sh
git clone https://github.com/Skaylatk/LABESK
```
3. Entre na Pasta Criada
4. Depois entre na pasta `src`
5. Encontre o Arquivo `labesk.c` e digite o comando:
```sh
gcc labesk.c -o LABESK
```
